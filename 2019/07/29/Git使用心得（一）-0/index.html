关于丢弃最新提交

commit之后想撤销本次提交：

HEAD^ 表示HEAD 往回数一个位置的commit ：即HEAD^ 表示你要恢复到哪个 commit。因为你要撤销最新的一个commit，所以你需要恢复到它的父 commit，也就是 HEAD^

想丢弃的提交不是最新提交

假如有一个 commit ，你在刚把它写完的时候并没有觉得它不好，可是在之后又写了几个提交以后，你想撤销了，不是最新的提交，就不能用 reset –hard 来撤销了。

如果想撤销第二条commit，那么可以使用rebase -i
